# Linux Check

A simple Bash CLI tool for checking system information.

## Features

- Storage information
- RAM usage
- Kernel version
- System uptime
- System update (Arch based)
- Interactive menu

## Requirements

- Bash
- Linux
- pacman (for system updates)

## Installation

Clone the repository:

```bash
git clone https://github.com/NikoHolm/linux-check.git
```

Go to the project:

```bash
cd linux-check
```

Make the script executable:

```bash
chmod +x linux.check.sh
```

Run:

```bash
./main.sh
```
